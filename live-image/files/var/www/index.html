ICPC-live autoanalyst

This host is running a ICPC-live system that contains an (almost) ready-to-use installation of the ICPC-live autoanalyst software toolset. It runs on the Debian GNU/Linux operating system and the Apache webserver.

Web interface

The autoanalyst web interface provides various views on the contest. A Gitweb interface for viewing the team homedirs is available if the code analyzer is in use.

Additionally phpMyAdmin is available. Login with the MySQL root user to access and modify the ICPC-live database named icat.

Note that initially all passwords are set equal to icpclive. See the script icpclive available on the system to change passwords.
